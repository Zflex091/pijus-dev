# Pijus.dev

Profesionalus asmeninis paslaugų puslapis, sukurtas su Next.js 15, TypeScript, Tailwind CSS, Framer Motion, Lenis ir Lucide React.

## Paleidimas

```bash
npm install
npm run dev
```

Atidarykite `http://localhost:3000`.

## Redagavimas

Kontaktai, navigacija, paslaugos ir kainos keičiami faile `data/site.ts`.
